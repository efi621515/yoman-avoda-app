🔥 מדריך הגדרת Firebase

הבעיה: הנתונים לא מסתנכרנים בין המחשב לפון כי Firebase לא מוגדר

⚠️ חשוב: בלי הגדרת Firebase, הסנכרון לא יעבוד!

שלב 1: יצירת פרויקט Firebase

1. לך ל-Firebase Console

2. לחץ על "Create a project"

3. שם הפרויקט: yoman-avoda-sync

4. לחץ "Create project"

שלב 2: הפעלת Firestore

1. בפרויקט החדש, לך ל-"Firestore Database"

2. לחץ "Create database"

3. בחר "Start in test mode"

4. בחר מיקום: europe-west1

5. לחץ "Done"

שלב 3: קבלת מפתחות API

1. לך ל-"Project Settings" (⚙️)

2. בחר "General" tab

3. גלול למטה ל-"Your apps"

4. לחץ על אייקון Web ()

5. שם האפליקציה: yoman-avoda-web

6. לחץ "Register app"

7. העתק את הקוד שנוצר!

שלב 4: עדכון הקוד

החלף את ההגדרות ב-index.html בשורה 7802:

const firebaseConfig = { apiKey: "המפתח שלך כאן", authDomain: "yoman-avoda-sync.firebaseapp.com", projectId: "yoman-avoda-sync", storageBucket: "yoman-avoda-sync.appspot.com", messagingSenderId: "המספר שלך", appId: "המזהה שלך" };

שלב 5: הגדרת כללי אבטחה

1. לך ל-"Firestore Database" → "Rules"

2. החלף את הכללים:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

3. לחץ "Publish"

שלב 6: בדיקה

1. שמור את הקובץ index.html

2. העלה ל-Netlify

3. פתח את האפליקציה בפון

4. לך להגדרות → סנכרון

5. לחץ "סנכרן עכשיו"

✅ אחרי ההגדרה:

  • הנתונים יסתנכרנו אוטומטית בין המכשירים
  • כל מכשיר יראה את הנתונים של האחר
  • הסנכרון יעבוד בזמן אמת
  • האפליקציה תעבוד מושלם על כל המכשירים

  • בדוק שהכללים נשמרו
  • בדוק שהפרויקט פעיל
  • בדוק בקונסולה של הדפדפן
